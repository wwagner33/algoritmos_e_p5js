Material Didático Interativo com P5.js

Bem-vindo ao material didático interativo sobre programação usando JavaScript e P5.js. Este conteúdo é especialmente criado para ensinar e demonstrar conceitos básicos e intermediários de programação por meio de exemplos visuais interativos. Você aprenderá sobre vetores, algoritmos de ordenação, buscas e cálculo de distâncias usando gráficos dinâmicos e interações em tempo real.

Explicação dos Conceitos e Implementações em P5.js

Este material apresenta conceitos fundamentais de programação utilizando JavaScript e a biblioteca gráfica P5.js. Os exemplos abordam a criação de vetores, ordenação simples, busca linear e cálculo de menor distância entre pontos.

Importante

Para poder ver o código em seu navegador, utilize a ferramenta Inspecionar. No Chrome e Firefox, basta teclar F12.

Na aba Console, você verá o código JavaScript que está sendo executado. Você pode interagir com o código, alterando valores e vendo os resultados em tempo real.

Arrays e sua aplicação em Vetores para Computação Gráfica

Vetores são estruturas essenciais em programação para armazenar e manipular conjuntos de dados. No exemplo das bolinhas, usamos arrays para armazenar posições e velocidades. Cada bolinha tem sua posição e velocidade atualizadas continuamente, com colisões detectadas visualmente.

O código JavaScript utiliza createVector para criar vetores que representam posições e velocidades, e métodos como .add() para atualizar posições.

Ordenação Simples (Bubble Sort)

O algoritmo Bubble Sort organiza os elementos em ordem crescente ou decrescente comparando pares adjacentes e trocando-os se estiverem fora da ordem desejada. A visualização mostra claramente cada etapa de comparação e troca.

O código JavaScript usa arrays para armazenar os valores, visualizando-os como barras, destacando em vermelho os elementos atualmente comparados.

Busca Linear Simples

A busca linear verifica cada elemento de uma lista sequencialmente até encontrar um valor específico ou atingir o fim da lista sem encontrá-lo.

O código JavaScript realiza a busca visualmente, destacando o elemento sendo analisado em tempo real, facilitando o entendimento do processo.

Cálculo da Menor Distância entre Pontos (Força Bruta)

Este algoritmo identifica o par de pontos mais próximo entre vários pontos, verificando todas as combinações possíveis (método força bruta).

O código JavaScript implementa esta verificação usando objetos do tipo p5.Vector, destacando visualmente com uma linha vermelha o par mais próximo encontrado.
