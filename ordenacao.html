Ordenação Simples (Bubble Sort)

O algoritmo Bubble Sort organiza os elementos em ordem crescente ou decrescente comparando pares adjacentes e trocando-os se estiverem fora da ordem desejada. A visualização mostra claramente cada etapa de comparação e troca.

O código JavaScript usa arrays para armazenar os valores, visualizando-os como barras, destacando em vermelho os elementos atualmente comparados.
